📚 Book Store Analytics Dashboard

Overview

This project demonstrates an end-to-end data analytics workflow using PostgreSQL and Power BI. The goal was to analyze bookstore sales, inventory, customer behavior, and revenue trends through SQL queries and interactive visualizations.
